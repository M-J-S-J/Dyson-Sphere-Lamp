## 注意事项
- 模型在打印的时候需要调整路径缩小等参数，否则后期需要大量打磨
- 模型预留的pcb空间较小，只有12mm×12mm×5mm，需要谨慎设计电路（由于我设计的pcb出现问题，后期飞线很多，就不开源了）
- 模型环是对称的，这里只提供了一半，直接打印两份即可
